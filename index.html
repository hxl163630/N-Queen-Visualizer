N Queens Puzzle Visualizer

The n queens puzzle is the problem of placing n chess queens on an n by n chessboard so that no two queens threaten each other; thus, a solution requires that no two queens share the same row, column, or diagonal. For futher infomation, please check here

This application visualize the Depth-first search (DFS) algorithm which solve the n queens puzzle step by step with COOL animations.
